Posts your Tweets to WordPress.

== Description ==

Inserts post containing your Twitter tweet's content and medias to your WordPress Site.
This plugin uses Twitter API Bearer Token.
So to use this, send a request to make your Developer's Account and make your App first.
Then get the bearer token for your Twitter App and set it to your configuration file.

== Installation ==

1. Upload All files to the `/wp-content/plugins/` directory.
2. Copy `tweet2wp-config-sample.php` file to `tweet2wp-config.php` file.
3. Write your Twitter information and bearer token to the file and save it.
4. Activate the plugin through the 'Plugins' menu in WordPress.

The plugin automatically gets your tweets and posts it hourly.
